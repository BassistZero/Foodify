# Foodify

Foodify is a recipe app for iOS.

## General Tech Info
- SwiftUI
- Combine
- MVVM
- â‰¥ iOS 15
- Pods:
  - SwiftLint
  - SwiftGen

## Installation

```sh
cd desired_folder
git clone https://github.com/BassistZero/Foodify.git
cd Foodify
pod install
open Foodify.xcworkspace
```
